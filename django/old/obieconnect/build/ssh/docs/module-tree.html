Module Hierarchy

  • ssh: 'ssh' is a module for python 2.3 or greater that implements the SSH2 protocol for secure (encrypted and authenticated) connections to remote machines.
    • ssh.agent: SSH Agent interface for Unix clients.
    • ssh.auth_handler: AuthHandler
    • ssh.ber
    • ssh.buffered_pipe: Attempt to generalize the "feeder" part of a Channel: an object which can be read from and closed, but is reading from a buffer fed by another thread.
    • ssh.channel: Abstraction for an SSH2 channel.
    • ssh.client: SSHClient.
    • ssh.common: Common constants and global variables.
    • ssh.compress: Compression implementations for a Transport.
    • ssh.config: SSHConfig.
    • ssh.dsskey: DSSKey
    • ssh.file: BufferedFile.
    • ssh.hostkeys: HostKeys
    • ssh.kex_gex: Variant on KexGroup1 where the prime "p" and generator "g" are provided by the server.
    • ssh.kex_group1: Standard SSH key exchange ("kex" if you wanna sound cool).
    • ssh.logging22: Stub out logging on python < 2.3.
    • ssh.message: Implementation of an SSH2 "message".
    • ssh.packet: Packetizer.
    • ssh.pipe: Abstraction of a one-way pipe where the read end can be used in select().
    • ssh.pkey: Common API for all public keys.
    • ssh.primes: Utility functions for dealing with primes.
    • ssh.resource: Resource manager.
    • ssh.rsakey: RSAKey
    • ssh.server: ServerInterface is an interface to override for server support.
    • ssh.sftp
    • ssh.sftp_attr
    • ssh.sftp_client: Client-mode SFTP support.
    • ssh.sftp_file: SFTPFile
    • ssh.sftp_handle: Abstraction of an SFTP file handle (for server mode).
    • ssh.sftp_server: Server-mode SFTP support.
    • ssh.sftp_si: SFTPServerInterface is an interface to override for SFTP server support.
    • ssh.ssh_exception: Exceptions defined by ssh.
    • ssh.transport: Transport handles the core SSH2 protocol.
    • ssh.util: Useful functions used by the rest of ssh.
    • ssh.win_pageant: Functions for communicating with Pageant, the basic windows ssh agent program.
